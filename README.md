### How to start

1. BE
   1.1. config => setup database.yml
   1.2. bundle
   1.3. rails db:migrate
   1.3. rails db:seed
   1.4. rails s

2. FE
   2.1. yarn install
   2.2. yarn run dev
   - will auto run "yarn run types" for graphQL types so make sure that BE is running (1.4.)
     or it will throw a error
